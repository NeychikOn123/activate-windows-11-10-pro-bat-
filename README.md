# Windows 11/10 Pro Activation Script

This batch file provides a complete solution for activating Windows 11 or Windows 10 Professional edition using KMS activation.

## Features
- Automatic administrator rights check
- Proper error handling
- Multiple activation methods as fallback
- Clear step-by-step progress reporting
- Activation status verification

## Requirements
- Windows 10/11 Pro (required)
- Administrator privileges
- Internet connection

## How to Use
1. Right-click on `activate_windows.bat` 
2. Select "Run as administrator"
3. Follow the on-screen instructions
4. The script will automatically attempt activation

## What the Script Does
1. Verifies administrator privileges
2. Installs the Windows Pro KMS client key
3. Configures the KMS host
4. Attempts activation
5. Provides alternative activation method if needed
6. Shows activation status

## Important Notes
- This is for educational purposes only
- Make sure you have proper licensing for Windows
- The activation may need to be renewed every 180 days with KMS activation
